# Comfort-Food

Food recipe app for people that love delicious and tasty foods.  This app enables users to search for a specific dish and it will search through the Edamam API for recipes.

## Technologies Used

### HTML
### CSS
### Bootstrap
### Vanilla Javascript
### Edamam API
